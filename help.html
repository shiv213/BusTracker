Help:

To update daily bus positions:

1. Go to the Account page (admin panel)

2. Click Bus Editor

3. Type in the bus number in each correct position's text box

4. If ready to send to students, double check buses and make sure the "Last Sent" isn't today, then press "SEND SMS" and follow confirmation dialog.

To mark temporary buses, add a minus symbol to the front of the bus number, and write the announcement to be made in the notes section.
Ex.) 34 would become -34
